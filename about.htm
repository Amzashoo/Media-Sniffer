Media Sniffer
Media Sniffer enables you to scan a folder of your computer for audio files and tries to grab information about your songs from the MusicBrainz knowledge database.
Required software :
  • Python 2.7
  • Levensthein for Python.
  • Mutagen for Python.
  • Karrigell web server.
  • RDFLib for Python.
